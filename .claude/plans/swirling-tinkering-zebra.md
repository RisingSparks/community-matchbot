# Post Opt-Out Flag Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let users request removal of their post via email, and give moderators a CLI command to set a per-post `opted_out` flag that excludes it from matching.

**Architecture:** Add an `opted_out: bool` field directly on the `Post` model (reversible, no data loss). Queue filters exclude opted-out posts. The forms thanks page gains an "email us" note. A new CLI command `matchbot posts opt-out` (and `un-opt-out`) sets the flag.

**Tech Stack:** SQLModel + SQLite, Alembic migrations, Typer CLI, inline HTML forms (no external templates)

### Task 3: Add CLI `posts opt-out` and `posts un-opt-out` commands

```python
@posts_app.command("opt-out")
def opt_out(post_id: str = typer.Argument(..., help="Post ID or unique prefix")) -> None:
    """Mark a post as opted out — excluded from matching."""

    async def _run() -> None:
        async with get_session() as session:
            post = await _resolve_post(session, post_id)
            if post.opted_out:
                typer.echo(f"Signal {post.id} is already opted out.")
                return
            post.opted_out = True
            await _write_event(session, post, "opted_out", note="User requested removal")
            await session.commit()
            typer.echo(f"Signal {post.id} opted out → excluded from matching.")

    asyncio.run(_run())


@posts_app.command("un-opt-out")
def un_opt_out(post_id: str = typer.Argument(..., help="Post ID or unique prefix")) -> None:
    """Restore a post that was previously opted out."""

    async def _run() -> None:
        async with get_session() as session:
            post = await _resolve_post(session, post_id)
            if not post.opted_out:
                typer.echo(f"Signal {post.id} is not opted out.")
                return
            post.opted_out = False
            await _write_event(session, post, "un_opted_out", note="Opt-out reversed")
            await session.commit()
            typer.echo(f"Signal {post.id} opt-out removed → back in matching pool.")

    asyncio.run(_run())
```

(Follow the exact import/helper patterns used by `dismiss` — `_resolve_post`, `_write_event`, `get_session`, `asyncio.run`.)
